Modulo: simons_core/__init__.py

1) Objetivo
Definir la superficie publica del paquete `simons_core` (exports, version y metadata base) para consumo consistente desde el resto del repositorio.

2) Alcance
Incluye:
- re-export de modulos core estables.
- definicion de `__version__` y metadata minima.
- politica de compatibilidad de API core.

3) Exports sugeridos
- interfaces
- schemas
- calendar
- logging
- io.paths
- io.parquet_store
- io.cache
- math.stats
- math.robust
- math.optimization

4) Reglas
- solo exportar APIs estables.
- cambios breaking en exports requieren version major.
- evitar imports costosos en tiempo de importacion del paquete.

5) Metadata recomendada
- __version__
- __all__
- __author__ (opcional)

6) Validaciones
- import simons_core funciona sin side effects no deseados.
- exports en __all__ resolubles.

7) DoD
- paquete core inicializa limpio y con API publica clara.
