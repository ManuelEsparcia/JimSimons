\section{Objetivo y Contexto Estratégico}
El módulo define la interfaz pública del paquete \texttt{simons_core}, re-exportando APIs estables para un ecosistema de swing trading cuantitativo ultra-rentable. Inspirado en Renaissance Technologies (Jim Simons), prioriza eficiencia en imports (evitando overhead $>1ms$) para backtests de $>10^6$ iteraciones diarias.

\textbf{Rentabilidad objetivo:} Integración modular permite Sharpe Ratio $>2.5$ en swing (holding 3-10 días), explotando mean-reversion via kernel methods \cite{simons_strategy}.

\section{Matemáticas Avanzadas en Submódulos Exportados}
\subsection{Estadística Robusta (math.robust)}
Usa estimadores breakdown-point $0.5$ para outliers en precios:
\[
\hat{\mu} = \median(\mathbf{x}), \quad \hat{\sigma} = 1.4826 \cdot \median(|\mathbf{x} - \hat{\mu}|)
\]
\cite{robust_stats}. Mejora swing: Detecta regimes volátiles para Kelly sizing $f^* = \frac{\mu}{\sigma^2}$ \cite{kelly}.

\subsection{Optimización (math.optimization)}
Bayesiana para hiperparámetros (GP-UCB):
\[
\alpha_t = \arg\max_{\mathbf{x}} \mu_t(\mathbf{x}) + \kappa \sigma_t(\mathbf{x})
\]
\cite{snoek2012practical}. State-of-art: Integra con Ax/BoTorch para portfolio opt en swing (CVaR min $\alpha=0.95$).

\subsection{Estadística (math.stats)}
Momentum swing: $z$-score $\theta_t = \frac{\bar{r}_{t-k:t}}{\sigma_k}$, TopK via dropout \cite{stockformer}:
\[
\hat{\theta} = \sum_{i \in \topk} w_i \theta_i, \quad w_i \sim \text{Bernoulli}(p=0.8)
\]
