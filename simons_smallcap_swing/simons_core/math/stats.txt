Modulo: simons_core/math/stats.py

1) Objetivo
Ofrecer utilidades estadisticas canonicas para research, validacion y reporting con implementaciones reproducibles y consistentes.

2) Alcance
Incluye funciones de tendencia central, dispersion, correlacion, rolling stats, intervalos y metricas financieras basicas.

3) Funciones core sugeridas
- mean/std/var robust wrappers
- rolling_mean/rolling_std/rolling_zscore
- rank_percentile
- spearman_ic/pearson_corr
- winsorize
- sharpe/sortino/calmar
- drawdown_series/max_drawdown

4) Reglas numericas
- manejo explicito de NaN.
- ddof consistente.
- estabilidad numerica (clip eps, evitar overflow).

5) API
- funciones puras con tipos claros.
- vectorizadas (numpy/pandas) cuando posible.
- mismas convenciones de retorno en todo el repo.

6) Validaciones
- tests contra casos conocidos.
- invariantes (std>=0, corr in [-1,1]).

7) CLI/Test hook
python -m simons_core.math.stats --self-test

8) DoD
- toolkit estadistico reutilizable y consistente.
