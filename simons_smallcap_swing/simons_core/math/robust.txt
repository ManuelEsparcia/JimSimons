\section{Objetivo y Contexto en Swing Trading}
Proveemos métodos robustos ante outliers y heavy tails, omnipresentes en datos financieros (e.g., flash crashes, earnings jumps). En swing trading, normalización robusta de features previene overfitting en ML models para signals.

\textbf{Problema clásico:} Media/moda sensible a \( O(n) \) outliers; robust: breakdown point 50\% [Hampel et al., 1986].

\textbf{Referencias clave:}
\begin{itemize}
\item Huber (1964): M-estimators, loss \(\rho_\epsilon(r) = \begin{cases} r^2/2 & |r| \leq \epsilon \\ \epsilon |r| - \epsilon^2/2 & |r| > \epsilon \end{cases}\).
\item Tukey (1977): Biweight para high efficiency.
\item Libro: \textit{Advances in Financial Machine Learning} (López de Prado, 2018) -- Ch. 5 Robust metrics.
\item Paper: ``Robust Portfolio Optimization'' (Konno \& Yamazaki, 1991).
\end{itemize}

SOTA: Robust transformers para TS [Zhou et al., 2021]; robust covariance via Tyler's M-estimator.

\section{Alcance y Funciones Clave}

\subsection{robust\_center\_scale(x)}
Retorna $(\median(x), \MAD(x))$. MAD: $\median_i |x_i - \median(x)|$, escala \( \MAD = 1.4826 \MAD \) (consistente con $\sigma$ normal).

\subsection{robust\_zscore(x)}
\[
Z_i = \frac{x_i - \median(x)}{1.4826 \cdot \MAD(x)}
\]
Breakdown point 50\%, ideal para returns log (leptokúrtcos).

\subsection{trimmed\_mean(x, p=0.05), winsorized}
Trimmed: Remueve $p\%$ extremos bilateral. Winsor: Clamp a percentiles.
\[
\bar{x}_{\trim} = \frac{1}{n(1-2p)} \sum_{i=p(n-1)+1}^{(1-p)(n-1)} x_{(i)}
\]
Usado en risk: VaR robusto [Danielsson et al., 2005].

\subsection{huber\_weights(residuals, c=1.345)}
Pesos M-estimator:
\[
w_i = \begin{cases} 
1 & |r_i| \leq c \\
c / |r_i| & |r_i| > c 
\end{cases}
\]
$c=1.345$ para 95\% efficiency (normal).

Tukey biweight: $\rho(r) = (\max(0, 1 - (r/c)^2))^2 / 6$.

\textbf{Código Python industrial:}
\begin{lstlisting}[language=Python, basicstyle=\ttfamily\small]
import numpy as np
from scipy.stats import median_abs_deviation
from typing import Tuple

def robust_center_scale(x: np.ndarray) -> Tuple[float, float]:
    center = np.median(x)
    scale = median_abs_deviation(x, scale=1/1.4826)  # Consistent sigma
    return center, scale

def robust_zscore(x: np.ndarray) -> np.ndarray:
    center, scale = robust_center_scale(x)
    return (x - center) / scale

def huber_weights(residuals: np.ndarray, c: float = 1.345) -> np.ndarray:
    return np.clip(np.abs(residuals) / c, 1.0, None).astype(float) ** -1 * (np.abs(residuals) <= c)
\end{lstlisting}

\section{Casos de Uso en Pipeline Swing}
- Normalización features (vol, momentum) antes de LSTM/GBDT.
- Anomalías: Detecta data issues pre-backtest.
- Risk params: $\sigma_{robust}$ para position sizing (fixed fraction).

Ej: Returns diarios S\&P, 1\% outliers $\to$ media bias 20\%, robust $\sigma$ estable.

- Pequeñas muestras: $n=30$, estabilidad vs. std.
