Modulo: simons_core/math/robust.py

1) Objetivo
Proveer metodos robustos a outliers/heavy tails para estimacion y transformaciones en datos financieros ruidosos.

2) Alcance
Incluye:
- median/MAD utilities
- robust z-score
- trimmed/winsorized estimators
- Huber-like loss helpers
- outlier detection robusta

3) Funciones sugeridas
- robust_center_scale(x)
- mad(x)
- robust_zscore(x)
- trimmed_mean(x, p)
- huber_weights(residuals, c)
- tukey_biweight(...)

4) Reglas
- defaults conservadores para colas pesadas.
- parametros robustos configurables.
- salida compatible con stats.py y feature_qc.

5) Casos de uso
- normalizacion de features.
- deteccion de anomalias en QC.
- estimacion robusta de parametros de riesgo.

6) Validaciones
- estabilidad ante outliers sinteticos.
- consistencia en muestras pequenas.

7) CLI/Test hook
python -m simons_core.math.robust --self-test

8) DoD
- utilidades robustas listas para pipeline y validation.
