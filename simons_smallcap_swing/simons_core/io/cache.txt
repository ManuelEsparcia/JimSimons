Modulo: simons_core/io/cache.py

1) Objetivo
Implementar cache local/reusable para acelerar calculos repetitivos y lecturas costosas, manteniendo coherencia por clave versionada.

2) Alcance
Incluye:
- cache get/set/invalidate
- claves por hash de inputs/config
- TTL opcional
- backend local filesystem (extensible)

3) API sugerida
- cache_key(namespace, params_hash)
- get(key)
- set(key, value, metadata)
- exists(key)
- invalidate(key/prefix)
- cleanup(expired_only=True)

4) Reglas
- claves deterministas.
- metadata minima: created_ts, ttl, producer_run_id, schema_hash.
- invalidacion cuando cambia config/schema_version.

5) Casos de uso
- features intermedias costosas.
- joins de datasets estaticos.
- resultados de tuning parcial.

6) Salidas
- objetos/cache files en ruta controlada.
- indice de cache para inspeccion.

7) Observabilidad
- cache_hit_rate
- cache_miss_rate
- bytes_cached
- avg_lookup_latency

8) CLI/Test hook
python -m simons_core.io.cache --stats

9) DoD
- cache segura y trazable, sin resultados stale peligrosos.
