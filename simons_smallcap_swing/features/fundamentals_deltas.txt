Modulo: features/fundamentals_deltas.py

1) Objetivo
Construir features de cambio fundamental (QoQ/YoY/aceleraciones) para capturar dinamica contable y senales de inflexion.

2) Alcance
Incluye calculo de deltas absolutos/relativos y aceleraciones sobre features fundamentales core con alineacion PIT estricta.

3) Entradas
- fundamentals_core_features_path
- fundamentals_pit_path
- deltas_config_path
- run_id

4) Salidas
- features/fundamentals_deltas/date={yyyy-mm-dd}/part-*.parquet
- features/fundamentals_deltas/summary_{run_id}.json
- features/fundamentals_deltas/manifest_{run_id}.json

5) Features de deltas sugeridas
- fund_revenue_growth_qoq/yoy
- fund_margin_delta_qoq
- fund_leverage_delta_yoy
- fund_cashflow_delta_qoq
- fund_growth_acceleration

6) Reglas PIT
- comparar solo con observaciones historicas ya publicadas.
- no usar period_end futuro no disponible en asof.

7) Flujo
1. Cargar core fundamentals por fecha.
2. Construir lags PIT-consistentes.
3. Calcular deltas y aceleraciones.
4. Aplicar clipping/normalizacion opcional.
5. Persistir outputs.

8) Validaciones
- coherencia temporal de lags.
- sin leakage en joins.
- cobertura minima por delta.

9) CLI
python -m features.fundamentals_deltas ^
  --fundamentals-core-path features/fundamentals_core ^
  --config-path configs/features/fundamentals_deltas.yaml ^
  --run-id fundamentals_deltas_20260305_001
