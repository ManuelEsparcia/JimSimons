Modulo: features/interactions.py

1) Objetivo
Construir features de interaccion entre familias de senales para capturar efectos no lineales y dependencias contextuales.

2) Alcance
Incluye combinaciones controladas de features (producto, ratio, gated interactions) con regularizacion de complejidad.

3) Entradas
- base_features_path (price/liquidity/fundamental/microstructure)
- interactions_config_path
- run_id

4) Salidas
- features/interactions/date={yyyy-mm-dd}/part-*.parquet
- features/interactions/summary_{run_id}.json
- features/interactions/manifest_{run_id}.json

5) Interacciones sugeridas
- int_mom_x_liq
- int_quality_x_vol
- int_value_x_regime
- int_micro_x_turnover
- int_growth_x_size

6) Reglas
- limitar numero de interacciones para evitar multiple testing.
- usar solo features con coverage adecuada.
- aplicar estandarizacion posterior.

7) Reglas PIT
- interacciones solo entre features ya PIT-safe en la misma fecha.
- sin introducir info futura en gating variables.

8) Flujo
1. Cargar set base de features.
2. Seleccionar pares/grupos segun config.
3. Calcular interacciones.
4. Aplicar clipping y validaciones.
5. Persistir outputs.

9) Validaciones
- sin explosiones de escala (outliers extremos).
- cobertura minima por interaccion.
- colinealidad excesiva flagged.

10) CLI
python -m features.interactions ^
  --base-features-path features/store ^
  --config-path configs/features/interactions.yaml ^
  --run-id interactions_20260305_001

11) DoD
- interacciones utiles y controladas listas para modelado.
