Modulo: features/microstructure.py

1) Objetivo
Generar features de microestructura (liquidez instantanea, friccion y dinamica de corto plazo) relevantes para small caps.

2) Alcance
Incluye proxies de spread, gap, range intradia, amihud-like y variacion de volumen/precio de alta frecuencia diaria.

3) Entradas
- prices_adjusted_path (OHLCV)
- market_proxies_path (opcional)
- microstructure_config_path
- run_id

4) Salidas
- features/microstructure/date={yyyy-mm-dd}/part-*.parquet
- features/microstructure/summary_{run_id}.json
- features/microstructure/manifest_{run_id}.json

5) Features sugeridas
- micro_spread_proxy
- micro_gap_open
- micro_hl_range
- micro_amihud_proxy
- micro_volume_shock
- micro_return_reversal_1d

6) Reglas PIT
- solo info de la sesion actual y pasado.
- no utilizar cierre futuro.

7) Flujo
1. Cargar OHLCV.
2. Calcular features micro por simbolo/fecha.
3. Aplicar clipping/winsor opcional.
4. Persistir y reportar cobertura.

8) Validaciones
- features finitas.
- rangos plausibles por feature.
- cobertura minima por simbolo.

9) CLI
python -m features.microstructure ^
  --prices-path data/price/adjusted/mode=dual_output ^
  --config-path configs/features/microstructure.yaml ^
  --run-id microstructure_20260305_001

10) DoD
- micro-features listas para build_features y feature_qc.
