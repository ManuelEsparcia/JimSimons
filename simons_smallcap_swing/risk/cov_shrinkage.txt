\section{Objetivo y Alcance}
Estimar matrices de covarianza estables mediante shrinkage para robustez en optimización de cartera swing trading (holding 5-20 días). Mejora conditioning number \(\kappa(\Sigma) = \frac{\lambda_{\max}(\Sigma)}{\lambda_{\min}(\Sigma)}\) de \(10^6\) a $<100$, reduciendo pesos extremos y drawdowns $[web:25]$.

Alcance: Cov muestral, shrinkage (Ledoit-Wolf/OAS/custom), validación PSD, persistencia. Integra con factor models downstream.

\section{Teoría Matemática Fundamental}
Covarianza muestral: 
\[
\hat{\Sigma} = \frac{1}{T-1} (X - \bar{X})^T (X - \bar{X}), \quad X \in \mathbb{R}^{T \times N}
\]
Inestable por ruido en muestras finitas (curse of dimensionality, $N > T/5$).

Shrinkage (Ledoit-Wolf 2004):
\[
\hat{\Sigma}_{shrink} = (1 - \delta) \hat{\Sigma} + \delta \Sigma_{target}, \quad \delta \in [0,1]
\]
donde \(\delta\) data-driven minimiza $E[\|\Sigma_{true} - \hat{\Sigma}\|^2_F]$.

Targets:
- Identity: $\Sigma_{target} = I$ (diagonal uniforme).
- Constant corr: $\rho_{ij} = 0.3$ (empírico equities), $\sigma_{target} = \sqrt{\hat{\Sigma}_{ii}}$.
- Factor: $\Sigma_{target} = B \Sigma_F B^T$ (Grinold \& Kahn 2000).

OAS (Chen 2010): Optimiza $\delta$ vía approximated log-likelihood para muestras pequeñas.

PSD validación: Eigen-decomp $\Sigma = Q \Lambda Q^T$, clip $\lambda_i < 0$ a $\epsilon = 10^{-6}$.

\section{Análisis Detallado del Módulo}
Flujo:
\begin{enumerate}
\item Load returns $X$ (parquet, lookback 252-1260 días, min\_obs=30).
\item \(\hat{\Sigma}_{sample} =\) np.cov.
\item Target: identity/constant\_corr/factor.
\item \(\delta =\) ledoit\_wolf (sklearn) o custom (rolling volatility regime).
\item \(\hat{\Sigma}_{shrink}\), validate simétrica/PSD (np.linalg.eigh).
\item Persist: cov\_shrunk.parquet, diagnostics.json (cond\#, $\delta$, forecast error $\| \hat{\Sigma} - \Sigma_{target} \|_F$).
\end{enumerate}

Casos especiales: Fallback identity si $T < 50$; jitter $\epsilon I$ si ill-conditioned.

\section{Optimizaciones SOTA para Swing Trading Rentable}
- Target híbrido: Constant corr + factor (Rajan \& Ray 2021 Hierarchical cov).
- \(\delta\) adaptativa: Alta en alta vol (VIX>25), baja en calm.
- Integra con GJR-GARCH para conditional cov: $\sigma_t^2 = \omega + \alpha \epsilon_{t-1}^2 + \gamma \epsilon_{t-1}^2 I_{\epsilon<0} + \beta \sigma_{t-1}^2$.
- ML boost: PCA eigenvalues para nonlinear shrinkage.

Código mejora (snippet):
\begin{lstlisting}[language=Python]
from sklearn.covariance import LedoitWolf
lw = LedoitWolf(assume_centered=False)
cov_shrunk = lw.fit(returns).covariance_
\end{lstlisting}
