\section{Objetivo Estratégico}
Este m\'odulo centraliza \emph{fixtures}, factories y configuraci\'on de \texttt{pytest} para tests reproducibles, r\'apidos y aislados en un pipeline de swing trading. Garantiza validaci\'on rigurosa de estrategias que capturan oscilaciones de 3-10 d\'ias, maximizando Sharpe ratio $\Sharpe = \frac{\E[R] - r_f}{\sigma(R)} > 2.5$ en backtests out-of-sample. Inspirado en reproducible research de Renaissance Technologies (RenTech), evita overfitting via seeds fijos y splits probabil\'isticos.[web:56][web:54]

\section{Alcance y Componentes Clave}
Incluye fixtures para:
\begin{itemize}
\item \texttt{tmp\_project\_dir}: Directorios temporales aislados, limpiados via \texttt{scope='function'}.
\item \texttt{sample\_run\_context}: Contexto de ejecuci\'on con seeds, timestamps y hiperpar\'ametros.
\item \texttt{synthetic\_price\_panel}: Paneles OHLCV sint\'eticos via Geometric Brownian Motion (GBM): $dS_t = \mu S_t dt + \sigma S_t dW_t$, con $\mu=0.0005$, $\sigma=0.02$ para simular swings realistas.[web:45]
\item \texttt{synthetic\_feature\_panel}: Features TA-Lib (RSI, MACD) + ML (PCA, autoencoders).
\item \texttt{synthetic\_label\_panel}: Labels binarios/multiclase para swings: $\label_t = \sign(R_{t+1:t+h} - \threshold)$, $h=5$ d\'ias.
\item \texttt{synthetic\_splits}: Train/val/test via TimeSeriesSplit, purgando look-ahead bias.
\end{itemize}

\section{Matemáticas Avanzadas: Generación Sintética de Datos}
Para fixtures determin\'isticas, usamos procesos estoc\'asticos calibrados a datos hist\'oricos (e.g., SPY daily). Precio bajo GBM:
\begin{equation}
S_{t} = S_0 \exp\left( (\mu - \frac{\sigma^2}{2})t + \sigma W_t \right)
\end{equation}
donde $W_t \sim \mathcal{N}(0,t)$ con NumPy seed fijo. Features incluyen momentum $\alpha_k = \frac{P_t - P_{t-k}}{\sigma_k(P)}$, calibrado a papers como Stockformer (62\% accuracy en trend detection).[web:45]

Labels para swing: maximizando Information Coefficient $IC = \corr(\pred_t, fwd_r_t)$, con forward returns $r_{t:t+h} = \log(P_{t+h}/P_t)$. SOTA: Incorporar GARCH(1,1) para volatilidad $\sigma_t^2 = \omega + \alpha \epsilon_{t-1}^2 + \beta \sigma_{t-1}^2$.[web:49]

\section{Implementación Optimizada: Fixtures Extendidas}
Ejemplo fixture sint\'etica (a explotar en JimSimons):
\begin{lstlisting}
import pytest
import numpy as np
import pandas as pd
from pathlib import Path

@pytest.fixture(scope="session")
def synthetic_price_panel():
    np.random.seed(42)  # Reproducible
    dates = pd.date_range('2020-01-01', periods=5000, freq='D')
    returns = np.random.normal(0.0005, 0.02, len(dates))
    prices = 100 * np.exp(np.cumsum(returns))
    return pd.DataFrame({'Close': prices}, index=dates)
\end{lstlisting}
Reglas: Scope jer\'arquico ('function' para aislamiento), no I/O externo, comparaciones con \texttt{np.allclose(rtol=1e-5)} para num\'erica estable.[web:56]
