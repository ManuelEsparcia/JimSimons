\section{Objetivo Estratégico}
Verificar capas PIT evitan informaci\'on futura (look-ahead bias), respetando \texttt{asof}/\texttt{filing}/\texttt{availability} timing. Cr\'itico para swing trading: previene inflado artificial de IC $IC = \corr(\pred_t, r_{t+1:t+h})$, asegurando OOS Sharpe realista $>2.8$. Estándar en RenTech/DE Shaw.[web:54][web:56]

\section{Casos Mínimos y Assertions Temporales}
\begin{itemize}
\item \texttt{test\_no\_future\_dates\_in\_prices}: $\price.date \leq \asof.date$.
\item \texttt{test\_edgar\_acceptance\_before\_asof}: $\acceptance\_ts \leq \asof$.
\item \texttt{test\_universe\_membership\_respects\_list\_delist\_window}: Membership solo en ventana $[list\_date, delist\_date)$.
\item \texttt{test\_no\_universe\_current\_leakage\_into\_history}: No universos futuros en historia.
\end{itemize}
Assertions: $\max(\src\_ts - \decision\_ts) \leq 0$, joins preservan $|LHS| \times |RHS|$, flags leakage$=0$.

\section{Matemáticas Avanzadas: PIT y Bias Purging}
PIT join: Para cada $\asof_t$, datos con $\src\_ts \leq \asof_t$. Leakage medido como $Leak = \P(\src\_ts > \asof_t)$. Purged CV (López de Prado): Embargo $T_{emb} = h + m$, $h$=hold period (5 días swing). Universe leakage: $U_t = \{s \in S | list_s \leq t < delist_s \}$. SOTA: Combinar con survivorship bias correction via Heckman model $\Prob(survive) = \Phi(X\beta)$.[web:54][web:45]

Para EDGAR: $filing\_delay = \acceptance - quarter\_end \sim \mathcal{N}(40,10)$ días, filtrar delays $>90\%$ percentile.
