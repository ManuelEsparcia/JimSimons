\section{Introducción y Contexto Estratégico}
En sistemas de swing trading, datos degradados (e.g., late prices) erosionan edge: $\alpha_t = f(\mathbf{X}_t) - \lambda \|\mathbf{X}_t - \hat{\mathbf{X}}_t\|^2$, donde $\hat{\mathbf{X}}$ es baseline sano [web:47]. Este módulo gatea el pipeline diario, detectando freshness/schema/anomalías para backtests limpios (Sharpe >2, low bias) [web:49].

Integra con daily\_pipeline como pre-gate, bloqueando si health\_score\_global < 0.95, clave en prod como QuantConnect o Renaissance [web:51].

\section{Matemáticas Fundamentales del Monitoreo}
\textbf{Freshness}: $F = 1 - \frac{t_{now} - t_{last}}{ \tau_{expected} }$, $\tau$=cutoff (e.g., 1h post-close), fail si $F<0.9$.

\textbf{Completeness}: $C = 1 - \frac{N_{missing}}{N_{expected}}$, $N_{expected} = |\mathcal{U}| \times T$, $\mathcal{U}$=universe, $T$=days.

\textbf{Schema Drift}: Distancia Kolmogorov-Smirnov $D = \sup_x |F_n(x) - F_{ref}(x)|$, ref=histórico.

\textbf{Anomalías}: Z-score $Z = \frac{x - \mu}{\sigma}$, outlier si $|Z|>3$; o Mahalanobis $\chi^2 = (\mathbf{x}-\boldsymbol{\mu})^T \Sigma^{-1} (\mathbf{x}-\boldsymbol{\mu})$.

\textbf{Health Score}: $H_d = \sum w_i h_i$, $\sum w_i=1$, e.g., $w_{fresh}=0.4$; global $H_g = \sum c_d H_d / \sum c_d$, $c_d$=criticidad (price=10x features).

Gates: Bayesian P(fail|data) >0.05 $\implies$ alert [web:54].

\textbf{Flujo Pseudocódigo Optimizado}:
\begin{lstlisting}[language=Python]
def main(config_path, run_id, exec_date):
    cfg = load_config(config_path)
    datasets = discover_datasets(cfg.catalog)
    healths = []
    for ds in datasets:
        checks = {
            'freshness': check_freshness(ds, exec_date),
            'completeness': check_completeness(ds),
            'schema': check_schema_drift(ds, ref_schema),
            'anomaly': check_anomalies(ds, method='mahalanobis')
        }
        H_ds = compute_score(checks, cfg.weights)
        status = 'pass' if H_ds > cfg.threshold else 'fail'
        healths.append({'dataset': ds.name, 'score': H_ds, 'status': status})
    
    H_global = weighted_avg([h['score'] for h in healths], weights=cfg.criticidad)
    alerts = build_alerts(healths, H_global < 0.9)
    persist_parquet(healths + alerts, run_id)
    return {'global_score': H_global, 'gate': H_global > 0.95}
\end{lstlisting}

\textbf{Casos Especiales}: Holidays ajustan $N_{expected}$ vía calendario (e.g., NYSE).
