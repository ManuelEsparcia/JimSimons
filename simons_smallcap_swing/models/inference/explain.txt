Modulo: models/inference/explain.py

1) Objetivo
Generar explicaciones de predicciones para modelos lineales/GBDT/ensemble (globales y locales), facilitando auditoria, debugging y confianza operativa.

2) Alcance
Incluye:
- explicabilidad global (importancias, ranking de features).
- explicabilidad local por prediccion (contribuciones por feature).
- soporte SHAP/permutation/coef-based segun modelo.
- reportes versionados por run.

No incluye:
- entrenamiento de modelos.
- decision final de cartera.

3) Dependencias
Upstream:
- modelos registrados
- predicciones y features de inferencia
- metadata de feature dictionary

Downstream:
- research/research_pipeline evidence
- validation reports
- monitoring de comportamiento de modelo

4) Contrato de entrada
Entradas minimas:
- model_ref
- features_path
- predictions_path
- explain_config_path
- run_id

Parametros clave:
- explain_mode (global|local|both)
- method (shap|permutation|coef)
- sample_size_local
- top_k_features

5) Contrato de salida
- models/explain/{run_id}/global_importance.parquet
- models/explain/{run_id}/local_explanations.parquet
- models/explain/{run_id}/explain_summary.json
- models/explain/{run_id}/manifest.json

Campos minimos local_explanations:
- date
- symbol
- model_id
- y_pred
- feature_name
- contribution
- rank_abs_contribution

6) Metodologias
Modelos lineales:
- contribucion = coef * feature_value.

GBDT:
- SHAP values (TreeSHAP) preferente.
- permutation importance como fallback.

Ensemble:
- explicacion por componente base + combinacion final.

7) Reglas de reproducibilidad
- fijar seed de muestreo local explanations.
- registrar version de libreria explainability.
- guardar subset utilizado para explain local.

8) Flujo
1. Cargar modelo, features y preds.
2. Resolver metodo de explicacion.
3. Calcular importancias globales.
4. Calcular explicaciones locales (muestra o full).
5. Persistir reportes.

9) Validaciones
- contribuciones finitas.
- consistencia baseline + sum(contributions) approx y_pred (si metodo lo permite).
- cobertura minima de explicaciones locales.

10) CLI
python -m models.inference.explain ^
  --model-ref lgbm_main_v3 ^
  --features-path features/store/date=2026-03-05 ^
  --predictions-path models/predictions/lgbm/run_latest/scores.parquet ^
  --config-path configs/inference/explain.yaml ^
  --run-id explain_20260305_001

11) DoD
- explicaciones globales/locales trazables y accionables para auditoria/model debugging.
